Plugin Scroll To Element and Spy

Import Plugin

Para importar el plugin se debe incluir el fichero js del plugin a través de la etiqueta<script>.

Debe incluir el plugin al final de la etiqueta <body>.
<body>
  ...
  <script type="text/javascript" src="http://fullstacksoftsas.com/lib/js/plugin/jquery.scrollSpyTo_fss.js"></script>
</body>

Html

Para el correcto funcionamiento del plugin se debe crear una estructura html que esté oriendata hacia dos enfoques. El primero es hacia el Scroll Top y el segundo enfoque hacia Scroll Left.
<!-- Scroll Top -->
<div class="contenedor">
  <ul>
    <li><a data-fss="item1" href="#">Item1</a></li>
    <li><a data-fss="item2" href="#">Item2</a></li>
    <li><a data-fss="item3" href="#">Item3</a></li>
    <li><a data-fss="item4" href="#">Item4</a></li>
    ...
    <li><a data-fss="itemN" href="#">ItemN</a></li>
  </ul>
  <section id="item1">Element 1</section>
  <section id="item2">Element 2</section>
  <section id="item3">Element 3</section>
  <section id="item4">Element 4</section>
  ...
  <section id="itemN">Element N</section>
</div>

CSS

Hay que adicionar los estilos correspondientes al componente en el que se usará en plugin. Importante es aclarar que si el plugin se usaráa en una elemento que no es el body entonces hay que crear una clase o id que identifique el elemento.
.contenedor {
  width: 500px;
  height: 240px;
  overflow-y: scroll;
  overflow-x: hidden;
  margin: 0 auto;
}
.contenedor ul {
  position: absolute;
  padding: 5px;
  width: 483px;
  background-color: #563d7c;
}
.contenedor ul li {
  display: inline;
  padding: 5px;
}
.contenedor ul li a {
  text-decoration: none;
  padding: 5px;
}
.contenedor ul li.active a {
  color: #c7254e;
}
.contenedor section {
  color: #FFF;
  width: 100%;
  height: 300px;
}
.contenedor #item1 {
  margin-top: 30px;
  background-color: #00AEFF;
}
.contenedor #item2 {
  background-color: #292929;
}
.contenedor #item3 {
  background-color: #FF3500;
}

JavaScript

Código JavaScript de uso simple para el enfoque al Scroll Top. 
<script>
  $(document).ready(function (e) {
    $(".contenedor").scrollSpyTo_fss({
	  selectorItem: '.contenedor ul li a', // Selector para los elementos del menu
	  parentItem: 1,                       // Padre para clase .active queda 1 nivel arriba
	  container: '.contenedor',            // Selector para el contenedor donde se hace el evento de Scroll
	  buffer: 30,                          // Atributo referido al offset que ocupa el elemento del menu, ejemplo el menu ocupa 30px de height
	  scrollTo: true                       // Activación del evento ScrollTo
    });
  });
</script>
